# vscode-pydoctest

vscode extension for the [Pydoctest](https://github.com/jepperaskdk/pydoctest) python package.

## Features

Highlights pydoctest errors, including error-descriptions.

## Requirements

Install pydoctest.

    $ python3 -m pip install pydoctest
